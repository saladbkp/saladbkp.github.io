the program is like ./dwarf main
to check its ELF, header , and flag ..

BUT HOW ???
beatified source code

program analysis from GPT
opcode 81: store character in flag array
opcode 82: perform checking

HOW to check?

  • Reads two indices (v15 and v22) from flag.
  • Reads an operation byte (v26), which determines the check type:
    • 0: Addition check (v12 = v21 + v24)
    • 1: Subtraction check (v12 = v21 - v24)
    • 2: Multiplication check (v12 = v21 * v24)
    • 3: XOR check (v12 = v21 ^ v24)
  • Compares the result to a given byte, enforcing a condition on the flag characters.

SO by logic we need v15 (indices 1) and v22 (indices 2), v26 (operation), the result v12

DISCORD solution, gdb script to capture the VALUE
v15 (indices 1): 0x1505
v22 (indices 2): 0x152f
v26 (operation): 0x153b
v12 (result): 0x1576

FINAL script
